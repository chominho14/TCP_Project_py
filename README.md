## TCP Project 2 (mini project)

---

### 개발 환경

- language : Python

---

### 프로젝트 설명

- 여러 사용자가 입장하여 업다운 게임을 진행할 수 있다.

---

### 진행 방법

1. 사용자 이름 입력창에 원하는 사용자 이름을 입력한다.

2. 하단의 입력창에 1부터 50사이의 숫자를 입력한다.

3. 입력한 숫자보다 정답이 크면 UP, 작으면 DOWN이 가운데에 있는 TextArea에 입력된다.

4. 총 기회는 5번의 기회가 주어진다.
